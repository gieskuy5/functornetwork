## Installation

1. Clone this repository to your local machine:

   ```
   git clone https://github.com/gieskuy5/functornetwork
   ```

2. Navigate to the project directory:

   ```
   cd functornetwork
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Run script:
   ```
   python login.py
   ```
